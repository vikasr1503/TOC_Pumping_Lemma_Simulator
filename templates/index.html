What is the Pumping Lemma?

The Pumping Lemma for Regular Languages is a theorem used to prove that a language is not regular. It states that for any regular language, there exists a "pumping length" p such that any string w in the language with length ≥ p can be split into three parts: w = xyz, satisfying:

  • |xy| ≤ p : The prefix and pumped part together are shorter than or equal to p.
  • |y| > 0 : The middle part y cannot be empty.
  • xyiz ∈ L for all i ≥ 0 : You can "pump" y any number of times, and the resulting string must still be in the language.

How to use this tool: Use the config below to pick a language, a string w, and decompose it into x, y, and z. If we find an i where the pumped string breaks the language conditions, it proves the language is not regular!
